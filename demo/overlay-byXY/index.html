First fetch your map anchor points:

const mapAnchorPoints = await api.fetchMapAnchorPoints(locationID, floorID);

Adding an annotation:

Convert your x and y points to latitude and longitude. To get an accurate latitude and longitude you need to include the width and height of your map in addition to the x,y values you would like to convert. In this example you can view the latitude and longitude in the console.

const xyToLatLng = MeridianSDK.mapPointToLatLng(mapAnchorPoints, {x: 5269.126531688274, y: 3824.493588695426}, {width: floorData.width, height: floorData.height});

To confirm we have generated the proper latitude and longitute, we will convert our lat/lng to screen X and Y points that we can create an annotation for and update the map with an annotation:

const latLngToXy = MeridianSDK.latLngToMapPoint(mapAnchorPoints, xyToLatLng);

meridianMap.update({
annotations: [
{
type: "point",
x: latLngToXy.x,
y: latLngToXy.y,
size: 32,
backgroundColor: "red",
backgroundImage: MeridianSDK.placemarkIconURL("generic")
 },
 ],
 });
